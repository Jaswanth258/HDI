📐 Algorithmic Formulation

The model utilizes the learned weights and intercept to map development indicators into a continuous decimal output representing the country's development level.

HDI = 0.1453 + 0.0128*(EYS) + 0.0107*(MYS) + 0.0057*(Life_Exp) + 1.9e-6*(GNI) + 2.4e-5*(Country_Enc)
EYS: Expected Years Schooling
MYS: Mean Years Schooling
Life_Exp: Life Expectancy
GNI: GNI per Capita (PPP $)
